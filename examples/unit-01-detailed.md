**Goal**: Identify red colored objects

## Lesson 9: Mixed Color Review (ListeningFour)

**Type**: ListeningFour (4-option multiple choice)  
**Goal**: Distinguish between all learned colors

### Format:
Show an object, user chooses from 4 color options

### Questions:
1. **Image**: Red apple → Options: [Red✓, Blue, Yellow, Green]
2. **Image**: Blue sky → Options: [Green, Red, Blue✓, Yellow]
3. **Image**: Yellow banana → Options: [Blue, Yellow✓, Red, Green]
4. **Image**: Green grass → Options: [Red, Blue, Green✓, Yellow]
5. **Image**: Red car → Options: [Red✓, Green, Blue, Yellow]
6. **Image**: Blue ocean → Options: [Yellow, Green, Blue✓, Red]
7. **Image**: Yellow sun → Options: [Red, Blue, Yellow✓, Green]
8. **Image**: Green tree → Options: [Blue, Green✓, Yellow, Red]

### Images Needed:
- 8 clear object images
- 4 color swatch buttons (red, blue, yellow, green)

- Yellow (2 languages)
- Green (2 languages)
- Orange (2 languages)
- Purple (2 languages)

### Key Success Factors:

1. **Image Quality**:
   - High resolution (1024x1024px minimum)
   - Clear, unambiguous representation
   - Consistent lighting and style
   - Diverse representation

2. **Color Accuracy**:
   - Red = RGB(255, 0, 0) range
   - Blue = RGB(0, 0, 255) range
   - Yellow = RGB(255, 255, 0) range
   - Green = RGB(0, 255, 0) range
   - Orange = RGB(255, 165, 0) range
   - Purple = RGB(128, 0, 128) range

3. **Progression**:
   - Introduce 2 colors at a time
   - Practice pronunciation immediately after learning
   - Review mixed colors before introducing new ones
   - End with comprehensive review

4. **Engagement**:
   - Use everyday, recognizable objects
   - Vary contexts (nature, food, vehicles, etc.)
   - Include elements relevant to target age group
   - Make learning feel practical and applicable
